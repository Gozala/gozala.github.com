<span style="font-family:arial;">This post is about piece</span> of <span style="font-family:arial;">great free software, it also has an open source version
Here is a link to it >> <a href="http://www.virtualbox.org/">http://www.virtualbox.org/</a>

But sometimes it is complicated to make it work with USB devices the way you need.
I made some research to make my USB devices work inside virtual machines on my Ubuntu 7.10
If you have the same problems and want to make it work then this post is certainly for you.

</span><ol style="font-family:arial;"><li>First of all you have to install non open source version of it.
<a href="http://www.virtualbox.org/wiki/Downloads">you can see how</a></li><li>Open terminal and type
<span style="font-family:courier new;"><span style="font-size:85%;">sudo gedit /etc/init.d/mountdevsubfs.sh</span>
<span style="font-family:arial;">uncomment lines below</span>
<span style="color:rgb(0,102,0);font-family:courier new;font-size:85%;">#mkdir -p /dev/bus/usb/.usbfs</span><span style="font-size:85%;">
</span><span style="color:rgb(0,102,0);font-family:courier new;font-size:85%;">#domount usbfs "" /dev/bus/usb/.usbfs -obusmode=0700,devmode=0600,listmode=0644</span><span style="font-size:85%;">
</span><span style="color:rgb(0,102,0);font-family:courier new;font-size:85%;">#ln -s .usbfs/devices /dev/bus/usb/devices</span><span style="font-size:85%;">
</span><span style="color:rgb(0,102,0);font-family:courier new;font-size:85%;">#mount --rbind /dev/bus/usb /proc/bus/usb</span><span style="font-size:85%;">
</span><span style="font-family:arial;">Save file</span></span></li><li>Open terminal and type
<span style="font-size:85%;"><span style="font-family:courier new;">sudo gedit /etc/fstab</span></span>
add line below
<span style="color:rgb(0,102,0);font-family:courier new;font-size:85%;"># usbfs</span><span style="font-size:85%;">
</span><span style="font-family:courier new;font-size:85%;">none /proc/bus/usb usbfs devgid=46,devmode=664 0 0 </span><span style="font-size:85%;">
</span>and save</li><li>Restart computer
</li><li>Enable in the setting of virtualbox support for usb and have a fun</li></ol>
